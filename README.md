# Sales Analysis

# Description

This dataset contains detailed information about customers, products, and store operations, enabling analysis of customer demographics, purchasing behavior, product performance, and regional store insights.

# Columns Description
| Column Name | Description |
| ------- | ---------- |
| CustomerID |	Unique identifier for each customer |
|Name |	Customer full name |
|Age |	Customer age |
|Gender	| Gender of the customer |
| City	| Customer city |
| State	| Customer state |
| Country |	Customer country |
| Loyalty | Level	Customer loyalty category (e.g., Silver, Gold, Platinum) |
| Product | ID	Unique identifier for each product |
| Product Name |	Name of the product |
| Category	| Main product category |
| Sub Category |	Product sub-category |
| Brand	| Product brand name |
| Cost |	Cost of the product |
| Stock	Available | stock quantity |
| Store_ID	| Unique identifier for each store |
| Store_Name	| Name of the store |
| Region	| Store region |
| Store City |	City where the store is located |
| Store_Type |	Type of store (Online, Retail, Franchise, etc.) |

# Tools Used

Excel / SQL – Data cleaning and transformation

Power BI / Tableau – Dashboard visualization

# Key Insights

Most customers fall into specific age groups (e.g., 25–40), helping target marketing campaigns.

Higher loyalty level customers (Platinum) contribute disproportionately to sales.

Portable power bank dominate sales.

High-cost items may have fewer sales but contribute more to revenue.

Accessories in sub-categories gains more profit.

Revenue yields least from west region($0.27).

Bean, Jones and Park store outperform others showing location-specific demand.

Low stock levels in high-demand products can lead to missed sales opportunities.
