# Sales Performance Analysis

# Description

This dataset contains detailed information about customers, products, and store operations, enabling analysis of customer demographics, purchasing behavior, product performance, and regional store insights.



# Tools Used

Excel  – Data cleaning and transformation

Power BI  – Dashboard visualization

# Key Insights

Most customers fall into specific age groups (e.g., 25–40), helping target marketing campaigns.

Higher loyalty level customers (Platinum) contribute disproportionately to sales.

Portable power bank dominate sales.

High-cost items may have fewer sales but contribute more to revenue.

Accessories in sub-categories gains more profit.

Revenue yields least from west region($0.27).

Bean, Jones and Park store outperform others showing location-specific demand.

Low stock levels in high-demand products can lead to missed sales opportunities.
